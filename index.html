PROYETCOS

SISTEMA CONTROL DE RESGUARDOS

Es un sistema desarrollado para controlar el uso de equipos del almacen para poder saber que personas de la empresa tienen que articulos y en que lugar, para saber si el equipo esta dispobible o no lo esta para que las demas personas puedan usarlos, el sistema imprime reportes de resguardo y de entrega de equipos.

TECNOLOGIAS QUE SE USARON
CHAT

Chat en tiempo real usando laravel websockets y pusher para la comunicacion, para poder chatear con personas primero debes de registrarte en el sistema con un correo que este dispoble ya que primero se envia un correro de verificacion, una vez activado el usuario ya puede logearse para empezar a chatear con las personas registradas en el sistema, puedes ver los usuarios conectados y desconectados en tiempo real, tambien los mensajes entre amigos son en tiempo real. Este sistema fue adaptable especialmente para dispositivos moviles
